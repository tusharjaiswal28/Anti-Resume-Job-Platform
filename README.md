# Anti-Resume Job Platform

## Introduction

The **Anti-Resume Job Platform** is a modern frontend-based hiring platform that replaces traditional résumés with skill-based challenges, profile showcases, and practical submissions.  
It allows users to log in, post jobs, submit challenges, view dashboards, and interact with a clean and simple UI — all using HTML, CSS, and JavaScript.  

This project is designed for beginners and intermediate developers learning how to structure a frontend project with modular pages and localStorage-based mock data handling.

## Features

* Login UI (frontend-only mock)
* Post job functionality (localStorage)
* View job listings
* Candidate skill challenges & submissions
* User dashboard with activity sections
* Profile page for candidate details
* Smooth navigation across multiple pages
* Clean and responsive layout using pure HTML/CSS

## Design Decisions or Assumptions

* No backend — all data stored using `localStorage`
* Simple JavaScript modules for jobs, profile, challenges
* UI-first design to help beginners understand structure
* Easy to extend into a full MERN or Node.js/Express backend
* Multiple HTML pages instead of a SPA for clarity
* Mock login for demonstration (not secure)
